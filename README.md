#  Example App powered by Cellular JS

## Project Overview

It's Users CRUD with some other CRUD references.

## General functionality

- CRUD Users
- CRUD Departments
- CRUD Permissions
- Reference: `Users`:`Department` -> `Departments`:`Name`
- Reference: `Users`:`Permission` -> `Permissions`:`Name`
